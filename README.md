# iDraw

后台采用nodeJs，前台通过gulp实现打包压缩实现-你画我猜-网页版,系统支持手机端与电脑端，并且系统采取分区分房间

在线地址: http://39.108.120.228:3000 系统支持手机端与电脑端

### 项目技术栈
●服务端：Node+Express4.0+Socket.io+Redis
●前端：JavaScript+HTML5+CSS3+gulp
●数据库：MySQL
